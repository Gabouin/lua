# Contributing

## How to contribute

1. Fork the repo
2. Make your changes in `index.html`
3. Test it by opening the file in a browser
4. Open a pull request with a short description of what you changed

## What's welcome

- Bug fixes
- New tools (fill bucket, line tool, etc.)
- UI improvements
- Anything on the roadmap in the README

## Keep it simple

The whole app is one HTML file — keep it that way. No build steps, no dependencies.
